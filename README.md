# Clasificación de pacientes con diabetes mediante Machine Learning

Este repositorio contiene el código asociado al Trabajo Fin de Máster (TFM) titulado:

**“Clasificación de pacientes con diabetes mediante técnicas de aprendizaje automático”**

## Contenido del repositorio

- `tfm_diabetes_ml.py`: script principal en Python que implementa el preprocesamiento de datos, el entrenamiento de distintos modelos de aprendizaje automático y la evaluación de resultados.
- `diabetes.csv`: conjunto de datos utilizado en el estudio.

## Requisitos

El código ha sido desarrollado en Python y utiliza las siguientes librerías:

- pandas
- numpy
- scikit-learn

## Ejecución del código

Para reproducir los experimentos descritos en la memoria del TFM, sitúese en el directorio del repositorio y ejecute:

```bash
python tfm_diabetes_ml.py
